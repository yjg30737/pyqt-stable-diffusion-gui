# pyqt-stable-diffusion-gui
PyQt(+PySide) Stable Diffusion GUI

## Requirements
### PyQt, PySide related 
* PyQt5>=5.14
* PySide6
* qtpy
### Stable Diffusion & HuggingFace related
* diffusers>=0.17.1
* torch
* pillow
* transformers
* accelerate
* huggingface_hub

## How to Use
1. install cuda
2. install torch from <a href="https://pytorch.org/get-started/locally">here</a>
3. git clone ~
4. pip install -r requirements.txt
5. python main.py
